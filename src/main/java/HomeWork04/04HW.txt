1) Встановити SQL Server
   - Mac: https://www.youtube.com/watch?v=glxE7w4D8v8
   - Windows: https://www.dnnsoftware.com/docs/developers/setup/set-up-sql.html
2) Створити користувача для вашої бази
3) Для доступу до БД можна використовувати будь-який едітор, наприклад Workbanch, dBeaver, SqlLite і тд.
4) Створити базу даних
5) Створити 3 різні таблиці
6) Наповнити ці таблиці даними
7) Спробувати використати різні селекти, з різними параметрами (як мінімум 5). Приклади можна глянути в презентації
8) Виконати ті самі селекти в Java коді за допомогою JDBC
